# 情感人格测试网站

一个使用 Next.js + Tailwind CSS 创建的手机端优先情感人格测试网站。无需登录、支付或数据库，所有题目和结果都保存在本地代码中。

## 本地运行

```bash
npm install
npm run dev
```

打开浏览器访问：

```text
http://localhost:3000
```

## 常用命令

```bash
npm run lint
npm run build
npm run start
```

## 部署到 Vercel

推荐方式：

1. 将项目推送到 GitHub。
2. 登录 Vercel。
3. 点击 Add New Project，导入该 GitHub 仓库。
4. 保持默认 Next.js 配置，点击 Deploy。

CLI 方式：

```bash
npm i -g vercel
vercel --prod
```

部署完成后，Vercel 会提供一个可分享的线上访问链接。

## 说明

测试结果只用于自我觉察和情绪陪伴，不构成医学、心理诊断或治疗建议。
